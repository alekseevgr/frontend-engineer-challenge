# Engineer Challenge

Full-stack приложение для аутентификации и авторизации пользователей.

Проект разделен на две части:

- `frontend/` - React + TypeScript клиент на Vite.
- `backend/` - REST API сервис на Rust, форк от [SilverHof/engineer-challenge](https://github.com/SilverHof/engineer-challenge).
